# ClawFounder｜面向 OPC 的 AI 创业流水线

比赛项目说明站（GitHub Pages 版本）。

## 项目定位

- 项目名：`ClawFounder`
- 副标题：`面向 OPC（One-Person Company）的 AI 创业流水线`
- 一句话简介：ClawFounder 将点子讨论、白皮书、原型开发、GitHub 建仓、网站上线、内容推广，以及通过 ADB 接入 root 手机实现的短信验证码接收与扫码登录辅助，收敛为一条统一的项目闭环。

## 比赛提交建议

### `project.title`

`ClawFounder｜面向 OPC 的 AI 创业流水线`

### `project.description`

ClawFounder 是一套面向 OPC（One-Person Company）的 AI 创业流水线。它解决的问题不是“不会写代码”，而是 OPC 很难低成本、持续地把想法做出来并推向市场。系统会先围绕一个新点子完成需求讨论、定位判断和白皮书生成；再将方案交给编码代理做出可运行原型，并在 GitHub 中建立项目资产；随后继续补齐品牌视觉、项目说明页、官网与上线入口。在项目上线后，系统还能根据会议录音、论坛纪要和外部信息生成文章、脚本、推广素材和阶段汇报，形成内容飞轮。更关键的是，ClawFounder 通过 ADB 接入 root 手机，把短信验证码接收与扫码登录辅助接入统一流程，补齐真实世界中最常见的自动化断点。最终，它把过去需要产品、设计、开发、运营共同完成的流程，压缩为 OPC 可持续运行的一条项目闭环。
