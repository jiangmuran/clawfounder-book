# ClawFounder｜会创业的龙虾

一个面向比赛提交的中文单页静态项目说明书站点。

## 项目定位

- 项目名：`ClawFounder`
- 副标题：`会创业的龙虾：把一个人的创业流程编译成 AI 流水线`
- 一句话简介：输入一个点子，它会完成方案定义、原型开发、品牌包装、网站上线、内容推广和阶段汇报。

## 页面内容

站点围绕这条叙事展开：

> OpenClaw 爆火之后，我最先担心的不是它不够强，而是它已经太能动了。
> 所以我没有继续追求一个“更强的助手”，而是把它训练成了一个有边界、能落地、会推动项目生命周期的一人创业操作系统。

页面包含：

- Hero 主叙事
- 问题提出
- 为什么不是普通 OpenClaw 案例
- 完整流水线
- 四个核心模块
- 两个典型运行场景
- 价值与效果
- 结尾提交提示

## 本地使用

直接双击 `index.html` 或使用任意静态文件服务打开即可。

## 比赛提交建议

### `project.title`

`ClawFounder｜会创业的龙虾：把一个人的创业流程编译成 AI 流水线`

### `project.description`

ClawFounder 是一只面向独立开发者与个人创作者的 AI 创业龙虾。它解决的问题不是“不会写代码”，而是“一个人很难低成本、持续地把想法做出来并推向市场”。当我有一个新点子后，ClawFounder 会先与大模型讨论需求和定位，生成白皮书与产品方案；再将方案交给代码代理生成可运行原型并推送到 GitHub；随后继续完善项目，自动制作 Logo、生成官网、部署页面并绑定域名。项目上线后，它还会制定推广计划，批量生成 GitHub issue 外联思路、公众号草稿、小红书软文和自媒体脚本，并定期向我汇报项目进展。除“从 idea 到上线”外，这只龙虾还能处理会议录音、论坛纪要等外部输入，自动总结核心观点，提炼成文章、选题和传播素材，反向喂给下一轮产品孵化。它把过去需要产品、设计、开发、运营共同完成的流程，压缩成一个人也能持续运行的创业流水线。

## 3 分钟视频建议

- `0:00 - 0:20`：抛问题——为什么一个人做项目总是烂尾
- `0:20 - 0:50`：输入一个 idea，产出白皮书
- `0:50 - 1:20`：生成原型、创建 GitHub 仓库
- `1:20 - 1:50`：补齐 Logo、官网、项目说明站
- `1:50 - 2:20`：生成公众号/小红书/推广计划
- `2:20 - 2:45`：展示会议纪要 → 观点 → 内容草稿 → 新选题
- `2:45 - 3:00`：收束——不是做一个助手，而是做一个人的创业操作系统
